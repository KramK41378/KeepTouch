# KeepTouch -
- WEB мессенджер, работающий на вычислительных мошностях компьютера пользователя.

# Описание
§ заходя на сайт пользователь проходит регистрацию, создает аккаунт
§ пользователь (хост) создает свою комнату и может приглашать в нее участников по айди, мессенджер работает на вычислиетльных мощностях хоста
§ пользователи могут принять приглашение и зайти в комнату для переписки
§ пользователи могут сохранять айди других пользователей с подписью (например чтобы добавлять в группу не запоминая айди пользователя), скорее всего сохраненную информацию о пользователях будем хранить в куки файлах самих пользователей
§ возможно добавим имена для пользователей (пользователь может написать себе ник)
